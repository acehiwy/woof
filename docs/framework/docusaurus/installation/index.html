Installation

Requirements​

  • Node.js version >= 16.14

Create Docusaurus Project​

The easiest way to install Docusaurus is to use the command line tool.

You can run this command anywhere in a new empty repository or within an existing repository

It will create a new directory containing the scaffolded files.

npx create-docusaurus@latest my-website classic
note

npx create-docusaurus@latest <name of project> <theme name> [ --typescript ]

or use these command​

  • npm
  • yarn
  • pnpm
  npm init docusaurus
  yarn create docusaurus
  pnpm create docusaurus
tip

Run npx create-docusaurus@latest --help, or check out its API docs for more information about all available flags.

Project Structure​

<Project Directory>├── blog│   ├── 2019-05-28-hola.md│   ├── 2019-05-29-hello-world.md│   └── 2020-05-30-welcome.md├── docs│   ├── doc1.md│   ├── doc2.md│   ├── doc3.md│   └── mdx.md├── src│   ├── css│   │   └── custom.css│   └── pages│       ├── styles.module.css│       └── index.js├── static│   └── img├── docusaurus.config.js├── package.json├── README.md├── sidebars.js└── yarn.lock
  • /blog/ - Contains the blog Markdown files. You can delete the directory if you've disabled the blog plugin, or you can change its name after setting the path option
  • /docs/ - Contains the Markdown files for the docs. You can delete the directory if you've disabled the docs plugin, or you can change its name after setting the path option.
  • /src/ - Non-documentation files like pages or custom React components .
  • /src/pages - Any jsx tsx mdx file within this directory will be converted into a website page.
  • /static/ - Static directory. Any contents inside here will be copied into the root of the final build directory
  • /docusaurus.config.js - A config file containing the site configuration.
  • /sidebars.js - Used by the documentation to specify the order of documents in the sidebar

Running Dev Server​

npm run start

By default, a browser window will open at http://localhost:3000.

Build​

npm run build

Contents will be generated within the /build directory, which can be copied to any static file hosting service

Update Docusaurus Version​

yarn upgrade @docusaurus/core@latest @docusaurus/preset-classic@latest
